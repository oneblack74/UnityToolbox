## ⚙️ Requirements

- Unity `6000.0+`
- [UniTask](https://github.com/Cysharp/UniTask)

---

## 🔧 Installation

1. Open **Window → Package Manager**
2. Click **+** → **Add package from Git URL**
3. Paste the Git URL of the desired package
